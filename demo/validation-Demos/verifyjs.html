Verify.js Framework validation:

Relies on jQuery

Relies on Notify.js for success / fail messages




Form Definition:
< form>

Verify using JQuery:
$("form").verify();

Field is REQUIRED and NUMBER:
data-validate="required,number"

Field is for Currency:
data-validate="currency"

Verify field using RegEx
data-validate="regex({Exp},{errorMsg})"

Verify using Custom Function
data-validate="customFunction"

Start Date Field
data-validate= "date,dateRange#start"

End Date Field
data-validate= "date,dateRange#end"
